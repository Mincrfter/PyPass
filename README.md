# PyPass
A basic, but clean-looking password generator in Python

There are 2 tabs to make it all look nice:
The Main tab, where you generate your password and copy it,
and the Settings tab where you customize the password generation

Nothing will be generated if you DONT put in a length and at least 1 type of character for the password
